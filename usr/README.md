# Flutter Counter Demo

Flutter Counter Demo is a simple starter app generated with [CouldAI](https://could.ai). It demonstrates a basic stateful Flutter screen with a counter, a floating action button, Material theming, and route-based app startup.

## App Features

- Shows a home screen titled `Flutter Demo Home Page`
- Displays the current counter value
- Increments the counter when the floating action button is pressed
- Uses a Material Design color scheme generated from a deep purple seed color
- Hides the debug banner for a cleaner preview
- Starts through the `/` route using Flutter's `routes` configuration
