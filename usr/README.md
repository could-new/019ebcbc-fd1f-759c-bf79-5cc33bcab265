# Black Fabrication Grill App

A Flutter application showcasing a custom-painted fullscreen black fabrication grill pattern design.

## Features
- Fullscreen custom painted grill background
- Responsive across all platform sizes
- Aesthetic industrial black metal style

## Tech Stack
- Flutter
- Dart CustomPainter

## Setup
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
